# Nexus Sentinel

Nexus Sentinel is an authorized school endpoint monitoring and policy-enforcement system designed for school-owned Windows workstations in managed environments.

It is intended for:
- school lab monitoring
- policy violation detection
- contextual alerting
- evidence preservation for confirmed incidents
- administrative oversight on institution-owned devices

## What it does

- Monitors active window titles and browser content
- Uses layered detection and confidence scoring
- Supports temporary input restriction for confirmed policy violations
- Preserves evidence through local buffering and synchronized upload
- Works with school-managed infrastructure and administrative control

## What this is not

- Not for personal devices
- Not for unauthorized monitoring
- Not for covert use outside institutional policy
- Not a general-purpose consumer app
